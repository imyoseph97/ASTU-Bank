ASTU Bank — project layout (quick map for students & markers)
==============================================================

Root
----
  README.md          How to run and what the program does
  J.iml              IntelliJ module (source root = src/main/java)
  .idea/             IntelliJ project settings (ignore if you use another IDE)

  src/main/java/     All Java source — standard layout for coursework
    banking/
      Main.java      Program entry point
      model/         UserAccount, Transaction, TransactionType (data classes)
      service/       BankService (business rules)
      store/         BankDataStore (load/save serialized accounts)
      security/      Password hashing
      ui/            Swing windows and screens

  data/              Created/updated when you run the app
    bank_data.ser    Serialized account database (primary persistence)
    users/           Plain-text snapshot per user (export / inspection)

  docs/              PROJECT_LAYOUT.txt, ASTU_Bank_Code_Walkthrough.pdf,
                     generate_code_walkthrough_pdf.py (run to rebuild the PDF)
  scripts/           Optional build/run helpers for Windows


Run from project folder (see README or scripts/run.bat).
